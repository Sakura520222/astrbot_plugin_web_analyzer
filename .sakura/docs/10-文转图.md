# 文转图

> 来源: https://docs.astrbot.app/dev/star/guides/html-to-pic.html

---

> **TIP**: 为了方便开发，您可以使用 [AstrBot Text2Image Playground](https://t2i.astrbot.app/) 在线可视化编辑和测试 HTML 模板。

## 基本用法

AstrBot 支持将文字渲染成图片。

```python
@filter.command("image")  # 注册一个 /image 指令，接收 text 参数。
async def on_aiocqhttp(self, event: AstrMessageEvent, text: str):
    url = await self.text_to_image(text)  # text_to_image() 是 Star 类的一个方法。
    # path = await self.text_to_image(text, return_url=False)  # 如果你想保存图片到本地
    yield event.image_result(url)
```

## 自定义（基于 HTML）

如果你觉得上面渲染出来的图片不够美观，你可以使用自定义的 HTML 模板来渲染图片。

AstrBot 支持使用 `HTML + Jinja2` 的方式来渲染文转图模板。

```python
# 自定义的 Jinja2 模板，支持 CSS
TMPL = '''
<div style="font-size: 32px;">
<h1 style="color: black">Todo List</h1>

<ul>
{% for item in items %}
  <li>{{ item }}</li>
{% endfor %}
</div>
'''

@filter.command("todo")
async def custom_t2i_tmpl(self, event: AstrMessageEvent):
    options = {}  # 可选择传入渲染选项。
    url = await self.html_render(TMPL, {"items": ["吃饭", "睡觉", "玩原神"]}, options=options)
    # 第二个参数是 Jinja2 的渲染数据
    yield event.image_result(url)
```

这只是一个简单的例子。得益于 HTML 和 DOM 渲染器的强大性，你可以进行更复杂和更美观的设计。除此之外，Jinja2 支持循环、条件等语法以适应列表、字典等数据结构。

## 图片渲染选项 (options)

请参考 Playwright 的 screenshot API：

| 参数 | 说明 |
|------|------|
| `timeout` | 超时时间 |
| `type` | 图片类型 |
| `quality` | 图片质量 |
| `omit_background` | 是否省略背景 |
| `full_page` | 是否截取完整页面 |
| `clip` | 裁剪区域 |
| `animations` | 动画处理 |
| `caret` | 光标处理 |
| `scale` | 缩放 |
