# AI 功能

> 来源: https://docs.astrbot.app/dev/star/guides/ai.html

---

AstrBot 内置了对多种大语言模型（LLM）提供商的支持，并且提供了统一的接口，方便插件开发者调用各种 LLM 服务。

您可以使用 AstrBot 提供的 LLM / Agent 接口来实现自己的智能体。

> **NOTE**: 我们在 `v4.5.7` 版本之后对 LLM 提供商的调用方式进行了较大调整，推荐使用新的调用方式。新的调用方式更加简洁，并且支持更多的功能。当然，您仍然可以使用旧的调用方式。

## 获取当前会话使用的聊天模型 ID

> 在 v4.5.7 时加入

```python
umo = event.unified_msg_origin
provider_id = await self.context.get_current_chat_provider_id(umo=umo)
```

## 调用大模型

> 在 v4.5.7 时加入

```python
llm_resp = await self.context.llm_generate(
    chat_provider_id=provider_id,  # 聊天模型 ID
    prompt="Hello, world!",
)
# print(llm_resp.completion_text)  # 获取返回的文本
```

## 定义 Tool

AstrBot 支持通过 `@dataclass` 定义 LLM Tool：

```python
from dataclasses import dataclass
from astrbot.api.provider import ToolMetadata, ToolFunc

@dataclass
class BilibiliTool(ToolFunc):
    """搜索B站视频"""
    
    tool_name: str = "bilibili_search"
    description: str = "搜索B站视频"
    parameters: dict = None
    
    def __post_init__(self):
        self.parameters = {
            "type": "object",
            "properties": {
                "keyword": {
                    "type": "string",
                    "description": "搜索关键词"
                }
            },
            "required": ["keyword"]
        }
    
    async def call(
        self, context: ContextWrapper[AstrAgentContext], kwargs
    ) -> ToolExecResult:
        return "1. 视频标题：如何使用AstrBot\n视频链接：xxxxxx"
```

## 注册 Tool 到 AstrBot

在上面定义好 Tool 之后，如果你需要实现的功能是让用户在使用 AstrBot 进行对话时自动调用该 Tool，那么你需要在插件的 `__init__` 方法中将 Tool 注册到 AstrBot 中：

```python
class MyPlugin(Star):
    def __init__(self, context: Context):
        super().__init__(context)
        # >= v4.5.1 使用：
        self.context.add_llm_tools(BilibiliTool(), SecondTool(), ...)

        # < v4.5.1 之前使用：
        # tool_mgr = self.context.provider_manager.llm_tools
        # tool_mgr.func_list.append(BilibiliTool())
```

## 通过装饰器定义 Tool 和注册 Tool

除了上述的通过 `@dataclass` 定义 Tool 的方式之外，你也可以使用装饰器的方式注册 tool 到 AstrBot。如果请务必按照以下格式编写一个工具（包括函数注释，AstrBot 会解析该函数注释，请务必将注释格式写对）：

```python
from astrbot.api.provider import llm_tool

class MyPlugin(Star):
    def __init__(self, context: Context):
        super().__init__(context)
    
    @llm_tool(name="get_weather", description="获取指定城市的天气信息")
    async def get_weather(self, city: str) -> str:
        """
        获取指定城市的天气信息
        
        Args:
            city: 城市名称
        """
        # 实际查询天气的逻辑
        return f"{city}今天晴，温度25°C"
```

使用装饰器方式时，Tool 会在插件加载时自动注册到 AstrBot 中。
