# 发布插件到插件市场

> 来源: https://docs.astrbot.app/dev/star/plugin-publish.html

---

在编写完插件后，你可以选择将插件发布到 AstrBot 的插件市场，让更多用户使用你的插件。

AstrBot 使用 GitHub 托管插件，因此你需要先将插件代码推送到之前创建的 GitHub 插件仓库中。

## 发布步骤

1. 前往 [AstrBot 插件市场](https://plugins.astrbot.app)
2. 点击右下角的 `+` 按钮
3. 填写好基本信息、作者信息、仓库信息等内容
4. 点击 `提交到 GITHUB` 按钮
5. 你将会被导航到 AstrBot 仓库的 Issue 提交页面
6. 确认信息无误后点击 `Create` 按钮提交，即可完成插件发布

## 注意事项

- 确保插件仓库是公开的
- 确保 `metadata.yaml` 文件填写完整且正确
- 建议添加 `logo.png` 和 `display_name` 以提升展示效果
- 建议添加 `README.md` 说明插件功能和使用方法
