# 开发平台适配器

> 来源: https://docs.astrbot.app/dev/plugin-platform-adapter.html

---

如果你想接入一个 AstrBot 尚未支持的消息平台，可以通过开发平台适配器来实现。

## 基本结构

平台适配器需要继承 AstrBot 的 `Platform` 基类，并实现必要的方法：

```python
from astrbot.api.platform import Platform, PlatformMetadata
from astrbot.api.message_components import MessageChain
from astrbot.api.star import Platform

class FakeAdapter(Platform):
    def __init__(self, platform_config: dict, platform_settings: dict, event_queue: asyncio.Queue) -> None:
        super().__init__(event_queue)
        self.config = platform_config  # 上面的默认配置，用户填写后会传到这里
        self.settings = platform_settings  # platform_settings 平台设置。
    
    async def send_by_session(self, session: MessageSesion, message_chain: MessageChain):
        # 必须实现
        await super().send_by_session(session, message_chain)
    
    def meta(self) -> PlatformMetadata:
        # 必须实现，直接像下面一样返回即可。
        return PlatformMetadata(
            "fake",
            "fake 适配器",
        )

    async def run(self):
        # 必须实现，这里是主要逻辑。
        pass
```

## 示例：模拟消息平台

下面是一个完整的示例，模拟一个每 5 秒下发一条消息的平台：

```python
class FakeClient():
    '''模拟一个消息平台，这里 5 秒钟下发一个消息'''
    def __init__(self, token: str, username: str):
        self.token = token
        self.username = username
                
    async def start_polling(self):
        while True:
            await asyncio.sleep(5)
            await getattr(self, 'on_message_received')({
                'bot_id': '123',
                'content': '新消息',
                'username': 'zhangsan',
                'userid': '123',
                'message_id': 'asdhoashd',
                'group_id': 'group123',
            })
            
    async def send_text(self, to: str, message: str):
        print('发了消息:', to, message)
        
    async def send_image(self, to: str, image_path: str):
        print('发了消息:', to, image_path)
```

## run() 方法实现

```python
async def run(self):
    # 必须实现，这里是主要逻辑。

    # FakeClient 是我们自己定义的，这里只是示例。这个是其回调函数
    async def on_received(data):
        logger.info(data)
        abm = await self.convert_message(data=data)  # 转换成 AstrBotMessage
        await self.handle_msg(abm) 
    
    # 初始化 FakeClient
    self.client = FakeClient(self.config['token'], self.config['username'])
    self.client.on_message_received = on_received
    await self.client.start_polling()  # 持续监听消息，这是个堵塞方法。
```

## 关键方法说明

| 方法 | 必须 | 说明 |
|------|------|------|
| `__init__` | ✅ | 初始化，接收 `platform_config`、`platform_settings`、`event_queue` |
| `send_by_session` | ✅ | 通过会话发送消息 |
| `meta` | ✅ | 返回适配器元数据 |
| `run` | ✅ | 适配器主逻辑，通常是监听消息的循环 |
| `convert_message` | ❌ | 将平台原始消息转换为 `AstrBotMessage` |
| `handle_msg` | ❌ | 将转换后的消息投入事件队列 |
